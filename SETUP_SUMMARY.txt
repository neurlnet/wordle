## ğŸ³ Docker Information

### Quick Docker Commands

```bash
# Build and start with hot-reload
docker-compose up --build

# Run without rebuilding
docker-compose up

# Run in background
docker-compose up -d

# View logs
docker-compose logs -f

# Stop
docker-compose down

# Full rebuild
docker-compose down && docker-compose up --build
```

### What Docker Does
1. Builds client with Vite (optimized)
2. Installs server dependencies
3. Runs both services concurrently
4. Mounts local code for hot-reload
5. Exposes ports 3001 & 5173
6. Health checks for monitoring
